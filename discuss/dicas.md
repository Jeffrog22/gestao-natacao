Para rodar este projeto, que é um template React + Vite, vocę precisará ter o Node.js instalado em sua máquina.

Siga os passos abaixo no seu terminal, dentro da pasta do projeto (c:\Users\HP\Desktop\Registro de Tempos\gestao-natacao):

Instale as dependęncias (caso ainda năo tenha feito):

bash
npm install
Inicie o servidor de desenvolvimento:

bash
npm run dev
Acesse no navegador: O terminal irá mostrar um link local, geralmente algo como http://localhost:5173. Basta clicar nele ou copiá-lo para o seu navegador para ver o projeto rodando.
